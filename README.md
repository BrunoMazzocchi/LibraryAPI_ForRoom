## API Reference
Common

#### Get all books

```http
  GET /books/all
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `No Parameter`      |  | **Required**. Search all books |


#### Get all borrows

```http
  GET /borrows/all
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `No parameter` |  | **Required**.  Gets all the borrows registered.
